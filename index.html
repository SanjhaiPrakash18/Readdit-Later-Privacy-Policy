1. Introduction

Welcome to Readdit Later, a Chrome extension designed to help you save, organize, and enhance your Reddit experience. This Privacy Policy explains how we collect, use, store, and protect your information when you use our extension and its AI-powered features.

Privacy-First Approach: Readdit Later prioritizes your privacy. Most data processing happens locally in your browser, and we only use external services with your explicit consent. You maintain full control over your data and privacy preferences.

2. Information We Collect

2.1 Reddit Account Information

When you connect your Reddit account via OAuth authentication, we collect:

  • Reddit Username: To identify your account and display your profile
  • OAuth Tokens: Access and refresh tokens for Reddit API access
  • Saved Posts: Your Reddit saved posts including titles, content, subreddits, scores, and metadata
  • Basic Profile Data: Public profile information as permitted by Reddit's OAuth scopes

2.2 Extension Usage Data

We store locally in your browser:

  • Your saved posts and organization preferences
  • Extension settings and customizations
  • Privacy preferences and consent choices
  • Synchronization timestamps
  • Local usage statistics (number of posts saved, subreddits accessed)

2.3 What We Do NOT Collect

  • Your Reddit password or login credentials
  • Private messages or direct communications
  • Posts you haven't explicitly saved
  • Your browsing history outside of Reddit
  • Personal identification beyond your Reddit username
  • Location data or device fingerprinting

3. AI Features and Data Processing

ðŸ¤– AI Features Require Consent: Our AI-powered features are entirely opt-in. You will be asked for explicit consent before any AI processing occurs, and you can disable these features at any time.

3.1 AI-Powered Features

With your consent, Readdit Later offers:

  • AI Summaries: Automatic post summarization using OpenAI/OpenRouter
  • Sentiment Analysis: Emotion and tone detection in posts
  • Auto-Labeling: Automatic categorization and tagging
  • Natural Language Search: Enhanced search capabilities
  • Subreddit Analysis: Community insights and statistics

3.2 AI Data Processing

When AI features are enabled:

  • Post content is sent to our secure server (reddit-later-server.vercel.app)
  • Our server processes requests using OpenRouter API with OpenAI's GPT-4o-mini model
  • Only post titles, content, and subreddit names are included
  • Personal identifiers and account information are automatically removed
  • Content over 1,500 characters is truncated to minimize data exposure
  • No data is permanently stored by AI providers or our server

3.3 Data Minimization

To protect your privacy, we implement data minimization:

  • Truncate long posts (over 1,500 characters)
  • Remove email addresses, phone numbers, and other personal identifiers
  • Exclude user comments and personal notes
  • Send only necessary context for AI processing

4. How We Use Your Information

4.1 Core Extension Functions

  • Display and organize your saved Reddit posts
  • Synchronize posts across browser sessions
  • Provide search and filtering capabilities
  • Generate usage statistics and insights
  • Enable import/export functionality

4.2 AI Enhancement (With Consent)

  • Generate summaries of lengthy posts
  • Analyze sentiment and emotional tone
  • Automatically categorize and label posts
  • Provide natural language search
  • Offer subreddit insights and trends

5. Third-Party Services

5.1 Reddit API

We use Reddit's official API for:

  • User authentication via OAuth 2.0
  • Retrieving saved posts and profile information
  • Performing save/unsave operations

Reddit API usage is governed by Reddit's Privacy Policy and API Terms.

5.2 AI Services (With Consent)

When AI features are enabled, we use:

  • OpenRouter API: For accessing OpenAI's GPT-4o-mini model for text processing and analysis
  • Our Secure Server: reddit-later-server.vercel.app processes AI requests without storing data
  • Privacy Controls: All AI processing includes automatic data sanitization

These services are used only with your explicit consent and are subject to their respective privacy policies.

5.3 Notion Integration (Optional)

If you choose to enable Notion integration:

  • Post data can be exported to your Notion workspace using our database template
  • OAuth authentication with Notion is required (Client ID: 275d872b-594c-805e-b3cc-00372d1e143b)
  • Data sharing is limited to your selected posts and includes titles, content, URLs, and metadata
  • You can revoke access through Notion's integration settings at any time

6. Data Storage and Security

6.1 Local Storage

Primary Storage: All your personal data is stored locally in your browser using Chrome's secure storage APIs. This means your data never leaves your device unless you explicitly choose to use AI features or export to third-party services.

6.2 Security Measures

  • OAuth 2.0: Secure Reddit authentication without password storage
  • HTTPS Encryption: All external communications use encrypted connections
  • Token Security: Secure token storage and automatic refresh
  • Content Security Policy: Strict CSP prevents unauthorized script execution
  • Rate Limiting: API request throttling prevents abuse
  • Minimal Permissions: Only necessary browser permissions requested

6.3 Server Processing (AI Features Only)

When using AI features:

  • Data is transmitted securely via HTTPS
  • Our server processes requests without permanent storage
  • AI providers receive only sanitized, anonymized content
  • No user identification or account data is shared

Technical Details

Chrome Permissions Explained

Our extension requests these specific Chrome permissions:

  • storage: Store your data and preferences locally
  • tabs: Open Reddit links and manage extension interface
  • alarms: Schedule background synchronization with Reddit
  • identity: Handle secure Reddit OAuth authentication
  • sidePanel: Display the extension interface in Chrome's side panel

Host Permissions

We request access to these specific domains:

  • https://www.reddit.com/*: Interact with Reddit website and enhance browsing
  • https://oauth.reddit.com/*: Handle Reddit OAuth authentication
  • https://api.reddit.com/*: Access Reddit's API for saved posts
  • https://reddit-later-server.vercel.app/*: Our server for AI processing (with consent)
  • https://api.notion.com/*: Notion integration (if enabled)
  • https://api.openai.com/*: OpenAI API access (via our server)
  • https://openrouter.ai/*: Alternative AI models (via our server)
Your Privacy Rights: You can revoke the extension's access to your Reddit account at any time through Reddit's authorized applications page. You can also disable AI features while keeping core functionality intact.
